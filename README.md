# crm-reparos

Aplicação desktop para gerenciamento de reparos eletrônicos, feita com JavaFX e Spring Boot.

O sistema permite cadastrar clientes, equipamentos, registrar ordens de reparo e acompanhar pagamentos — tudo com uma interface simples e conectada a uma API REST.

Ideal para oficinas e técnicos que precisam organizar o fluxo de serviços de forma prática e centralizada.
